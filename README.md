# SynapseLab

基于 React + CloudBase 的跨学科科研协作平台。

## 项目定位

SynapseLab 的目标是把研究者、猜想、项目、资源和 AI 助理连接起来，帮助跨学科科研从灵感走向验证。

## 当前阶段

当前仓库处于“项目骨架与文档规划”阶段，先完成：

1. 目录结构搭建
2. 文档分类整理
3. 前后端模块拆分
4. 本地开发方案
5. CloudBase 部署预留

## 目录说明

- `docs/`：论文文档、需求文档、设计文档、部署文档
- `web/`：React 前端
- `cloudfunctions/`：CloudBase 云函数，相当于后端
- `database/`：数据库设计、初始化数据、权限规则
- `tests/`：测试代码
- `deploy/`：部署说明、环境配置样例、上线清单
- `materials/`：原始需求、原型图、答辩材料

## 推荐开发顺序

1. 先阅读 `docs/00-项目总览.md`
2. 再阅读 `docs/01-需求分析.md`
3. 然后查看 `web/README.md` 和 `cloudfunctions/README.md`
4. 前期先做本地开发，后期再上 CloudBase

## 给你的理解版本

你可以把这个项目理解成三层：

- 前端：用户能看到和点击的页面
- 后端：处理数据和业务逻辑的云函数
- 文档：帮助你写论文、答辩、部署和后续维护
