# 截图素材说明

这个目录用于存放 GitHub 仓库展示用的真实页面截图。

建议后续放入 4 到 5 张截图：

- `home.png`
- `ideation.png`
- `studio.png`
- `network.png`
- `graph-mode.png`

推荐尺寸：

- 桌面端横图
- 宽度 1440 左右
- 保持统一比例和浏览器窗口样式

推荐展示内容：

- 首页核心视觉
- 思想熔炉列表或详情
- 研究工作室项目视图
- 学者网络关系图谱
- 全屏图谱模式

当前仓库还没有现成的真实截图文件。如果后续把截图补到本目录，README 就可以直接引用这些图片。
