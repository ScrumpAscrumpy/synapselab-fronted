# 项目总览

## 项目名称

- 中文名：SynapseLab 跨学科科研协作平台
- 英文名：SynapseLab
- 论文题目：基于 React + CloudBase 的跨学科科研协作平台 SynapseLab 的设计与实现

## 项目愿景

SynapseLab 希望解决跨学科科研协作中的知识孤岛问题，让一个猜想可以更快找到合作者、验证路径和执行环境。

## 核心模块

1. 首页
2. 思想熔炉
3. 研究工作室
4. 学者网络
5. AI 研究助理 Lumina

## 技术路线

- 前端：React
- 后端：CloudBase 云函数
- 数据层：CloudBase 云数据库
- 部署：CloudBase 静态托管 + 云函数部署

## 当前开发策略

1. 先搭项目结构
2. 先做本地可演示版本
3. 再接入 CloudBase
4. 最后整理论文和部署材料
