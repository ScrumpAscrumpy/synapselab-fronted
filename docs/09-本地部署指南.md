# 本地部署指南

## 这份文档是干什么的

教你如何在自己电脑上把项目跑起来。

## 当前状态

目前前端骨架已经完成，包含：

1. React 前端基础入口
2. 页面路由
3. 左侧导航布局
4. 首页
5. 思想熔炉页
6. 研究工作室页
7. 学者网络页

但你的电脑当前缺少 `Node.js` 和 `npm`，所以暂时还不能直接运行前端。

## 给小白的理解

“本地部署”就是先让项目在你自己的电脑上运行，不需要先上云。

## 你现在需要做什么

你只需要完成下面这几步：

1. 安装 Node.js
2. 打开终端
3. 进入项目目录
4. 安装前端依赖
5. 启动前端项目
6. 在浏览器里打开本地地址

## 第一步：安装 Node.js

### 最简单的方法

1. 打开 Node.js 官网
2. 下载 `LTS` 版本
3. 一路点击“继续”安装
4. 安装完成后，关闭终端，再重新打开一个新终端

## 第二步：检查是否安装成功

在终端里输入下面两条命令：

```bash
node -v
npm -v
```

如果都能显示版本号，就说明安装成功。

如果仍然提示 `command not found`，把报错发给我，我继续带你处理。

## 第三步：进入项目目录

在终端输入：

```bash
cd "/Users/scrumpy/Documents/New project/SynapseLab"
```

## 第四步：安装前端依赖

输入：

```bash
npm install --prefix web
```

这一步会自动安装 React、Vite、路由等前端依赖。

第一次安装通常会稍慢，这是正常的。

## 第五步：启动前端项目

输入：

```bash
npm run dev
```

如果启动成功，终端一般会出现本地访问地址，通常类似：

```bash
http://localhost:5173
```

## 第六步：打开浏览器查看页面

把终端里显示的网址复制到浏览器打开。

你应该能看到：

- 左侧导航栏
- 首页 Hero 区域
- 思想熔炉卡片
- 研究工作室模块
- 学者网络模块

## 如果页面打不开

优先检查这几件事：

1. `node -v` 和 `npm -v` 是否有版本号
2. 是否在 `SynapseLab` 根目录执行了命令
3. 是否执行过 `npm install --prefix web`
4. 终端是否有红色报错

## 当前前端命令说明

在项目根目录 `/Users/scrumpy/Documents/New project/SynapseLab` 下：

```bash

```

含义是：

- 自动进入 `web/`
- 启动 Vite 前端开发服务器

如果你想直接在前端目录运行，也可以：

