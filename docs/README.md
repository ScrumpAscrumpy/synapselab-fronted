# docs

这个文件夹专门放“写文章”和“做系统说明”需要的文档。

## 你可以怎么理解

如果 `web/` 和 `cloudfunctions/` 是做项目本体，那么 `docs/` 就是帮你：

- 写毕业论文
- 做开题、中期、答辩材料
- 解释系统为什么这样设计
- 教你怎么部署

## 文档分类

- `00-项目总览.md`：整个平台做什么
- `01-需求分析.md`：用户需要什么、为什么做这个系统
- `02-系统架构设计.md`：前后端和云端怎么配合
- `03-数据库设计.md`：数据库结构
- `04-接口设计.md`：前后端如何通信
- `05-页面设计.md`：每个页面做什么
- `06-AI功能设计.md`：AI 助理怎么设计
- `07-积分与贡献度设计.md`：积分、贡献、激励逻辑
- `08-权限与安全设计.md`：权限、隐私、安全
- `09-本地部署指南.md`：如何在自己电脑跑起来
- `10-CloudBase部署指南.md`：如何部署到腾讯云 CloudBase
- `11-测试方案.md`：怎么测试系统
- `12-论文写作素材.md`：可直接转化为论文内容的材料
- `13-上线验收报告.md`：系统上线结果与问题处理记录
- `14-答辩展示说明.md`：答辩时怎么讲、怎么演示
- `15-毕业论文初稿.md`：基于本项目生成的论文初稿
- `16-毕业论文二稿.md`：更接近正式提交稿的论文版本
- `17-论文截图与插图清单.md`：论文截图和图表准备清单
- `18-论文提交检查清单.md`：提交前逐项核对
- `19-毕业论文三稿.md`：补充软件说明书内容后的增强版论文
- `20-毕业论文终稿版.md`：更适合直接整理进 Word 的最终文本版本
- `21-测试结果表.md`：论文和说明书可直接使用的测试表格
- `22-答辩PPT讲稿版.md`：可直接照着讲的答辩讲稿

## 建议阅读顺序

1. `00-项目总览.md`
2. `01-需求分析.md`
3. `02-系统架构设计.md`
4. `05-页面设计.md`
5. `10-CloudBase部署指南.md`
