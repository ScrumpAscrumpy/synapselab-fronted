# 部署分工说明

这份文档专门告诉你：部署 SynapseLab 时，哪些事情必须你自己做，哪些事情我可以继续帮你做。

## 你必须自己做的事

### 1. 腾讯云账号相关

- 注册账号
- 实名认证
- 登录控制台
- 创建 CloudBase 环境

原因：

这些操作和你的个人账号、权限、计费相关，必须由你自己确认。

### 2. 控制台里的关键确认

- 查看环境 ID
- 确认资源是否创建成功
- 确认是否在正确账号下操作

## 我可以帮你做的事

### 1. 代码和配置

- 改前端环境变量
- 改云函数代码
- 改数据库模型
- 改服务层接口
- 改部署说明文档

### 2. 排错

- 看报错日志
- 分析为什么部署失败
- 告诉你下一步该点哪里
- 在环境允许时直接执行命令

### 3. 联调

- 检查页面是不是连到了真实数据
- 检查云函数返回结构对不对
- 检查数据库字段是不是匹配前端

## 最推荐的协作方式

等正式部署时，建议你这样配合我：

1. 你开着腾讯云控制台
2. 我告诉你下一步点哪里
3. 你把看到的页面、环境 ID 或报错发给我
4. 我继续帮你判断和修改代码
